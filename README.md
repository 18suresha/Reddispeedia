# Reddispeedia

## Description

A Reddit version of [Wikispeedia](https://dlab.epfl.ch/wikispeedia/play/) using hyperlinks in subreddits to other subreddits.

## Presentation Video
https://youtu.be/SmvtDUWfLV8

## Running on a Local Machine
Build and run Reddi-Spedia
```bash
make
./main
```
Build and run tests
```bash
make test
./test
```
